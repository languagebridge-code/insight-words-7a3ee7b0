Quick Start Guides

Installing the Extension

Step-by-step guide to install LanguageBridge on Chrome:

  1. Open Google Chrome browser
  2. Visit the Chrome Web Store
  3. Search for 'LanguageBridge'
  4. Click 'Add to Chrome'
  5. Confirm by clicking 'Add Extension'
  6. The LanguageBridge icon will appear in your toolbar

Initial Setup

Configure your classroom and student languages:

  1. Click the LanguageBridge icon in Chrome
  2. Enter your teacher code provided by your school
  3. Select the primary languages your students speak
  4. Customize translation settings as needed
  5. Test with a sample webpage

Adding Students

Help students access LanguageBridge in class:

  1. Share your classroom code with students
  2. Students install the extension on their Chromebooks
  3. They enter the classroom code when prompted
  4. Monitor student activity from your dashboard

Troubleshooting

Extension Not Appearing in Toolbar

  • Click the puzzle icon in Chrome toolbar to find LanguageBridge
  • Pin the extension by clicking the pin icon
  • Try reinstalling the extension from Chrome Web Store
  • Restart Chrome browser

Extension Not Translating Text

  • Make sure you're on a supported website
  • Check if the extension is enabled (icon should be colored, not gray)
  • Refresh the webpage and try again
  • Check your internet connection
  • Verify your classroom code is entered correctly

No Audio Playback

  • Check if your device volume is turned up
  • Make sure audio is not muted in Chrome
  • Try using headphones
  • Check Chrome's site permissions for audio
  • Restart the browser

Can't Log In to Dashboard

  • Verify your email address is correct
  • Use the 'Forgot Password' link to reset
  • Check if your school account is active
  • Contact your school's IT administrator

Frequently Asked Questions

What browsers does LanguageBridge support?

LanguageBridge currently supports Google Chrome and Chromium-based browsers (Edge, Brave). We recommend using the latest version of Chrome for the best experience. Support for Firefox and Safari is planned for future releases.

Is LanguageBridge FERPA and COPPA compliant?

Yes! LanguageBridge is fully FERPA and COPPA compliant. We never store student data, all translations happen in real-time without saving personal information, and we have signed Data Protection Agreements with participating districts.

What languages does LanguageBridge support?

LanguageBridge currently supports Somali, Urdu, Ukrainian, Persian, Dari, Pashto, Arabic, Spanish, and English—with more languages coming soon based on user requests.

Can students use LanguageBridge on their personal devices?

Students can use LanguageBridge on any Chrome browser with their classroom code. This includes personal Chromebooks, laptops, and desktop computers. Mobile support is coming soon.

How does the audio feature work?

LanguageBridge provides native-speaker audio for all translations. Students can click the speaker icon to hear the pronunciation, which is especially helpful for preliterate students who are still developing reading skills in their home language.

What's included in the free pilot program?

The free pilot includes full access to all LanguageBridge features, professional development training for teachers, dedicated support, and monthly usage reports. We're offering this to select Ohio schools for the 2025-2026 school year.

How do I get usage reports for my classroom?

Teachers can access usage reports through the Teacher Dashboard. You'll see translation counts, language pairs used, and student engagement metrics. District administrators can access aggregate reports across all classrooms.

How do I report a translation error?

Click the feedback button on any translation to report errors. Our team of native speakers reviews all submissions and improves translations regularly. You can also email support@languagebridge.app with specific examples.
